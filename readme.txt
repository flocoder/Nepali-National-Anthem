This plugin adds Nepali National Anthem lyrics in WordPress dashboard.

== Description ==

This plugin adds Nepali National Anthem lyrics in WordPress dashboard. This plugin is forked from Hello Dolly plugin by Matt Mullenweg. When activated you will randomly see a lyric of Nepali national anthem (<cite>सयौं थुँगा फूलका</cite>) in the upper right of your admin screen on every page.
